Airbnb Clone

A full-stack Airbnb Clone that enables users to discover, list, and book vacation rentals with a modern, 
responsive interface. The application replicates the core functionality of Airbnb, including property listings, 
authentication, image uploads, and user reviews.



Features
  > Secure user authentication and authorization
  > Create, edit, and delete property listings
  > Add and manage reviews and ratings
  > Upload property images with Cloudinary
  > Save favorite listings to a wishlist
  > RESTful APIs with robust backend validation
  > Custom error handling and input validation

Tech Stack
  Node.js
  Express.js
  MongoDB
  Mongoose
  EJS
  Bootstrap
  Passport.js
  Cloudinary
